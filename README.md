# Download youtube videos for android
##download Termux app
```
apt update && apt upgrade
```
```
pkg update && pkg upgrade
```
```
git clone https://github.com/Professor606/youtube-vids.git
```
```
cd youtube-vids
```
```
chmod +x youtube.sh
```
##you can now use this tool by typing:
```
./youtube.sh
```
